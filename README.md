# Sistem-Inventaris-Toko
Sistem ini bertujuan untuk mengelola atok barang agar lebih efisien dan terdata rapi sehingga mudah untuk memantau flow barang.

Folder Model : Membuat Object
Folder View : Menampilkan UI
Folder Controller : Berisi reaksi yang muncul ketika user berinteraksi (tidak mengoperasikan data/perantara user dan program)
